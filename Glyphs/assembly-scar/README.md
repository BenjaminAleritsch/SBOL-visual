# Assembly Scar

## Associated SO term(s)
SO:0001953

## Recommended Glyph and Alternates
The assembly scar glyph is an "equal sign" image, the pattern produced by the union of a 5' sticky end and 3' sticky end glyph:

Scar on top of backbone:
